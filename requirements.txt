# Step-by-step commands to set up and run the FIFA Portfolio React project

# 1. Install Node.js (includes npm)
#    Use the installer from https://nodejs.org/ or a version manager like nvm:
#
#    curl -fsSL https://fnm.vercel.app/install | bash   # example using fnm
#    nvm install --lts                                 # example using nvm
#
#    Verify installation:
#    node --version
#    npm --version

# 2. Clone the repository (if not already done):
#
#    git clone <repo-url> fifa-portfolio
#    cd fifa-portfolio

# 3. Install project dependencies:
#
#    npm install
#
#    This command reads package.json and installs everything required.

# 4. Run the development server:
#
#    npm start
#
#    The app will be available at http://localhost:3000

# 5. Build for production (optional):
#
#    npm run build
#
#    The optimized files will be in the "build" directory.

# 6. Additional useful commands:
#
#    npm test      # run tests (if any)
#    npm run eject # eject CRA config (irreversible)
